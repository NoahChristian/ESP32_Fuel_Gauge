This is code for a Home Automation battery state of charge indicator. Using a 40 segment LED strip, each LED represents 2.5% on state of charge. 

Red signifies discharge level, blue is the charge level. 

During no activity (the battery is sitting idle) no animiations are made on the LED strip. If the battery is discharging, a sweep of LEDs goes downwards. If the battery is charging, it goes in the upward direction. Each animation color is slightly different, and can be programmed and currently hardcoded.
